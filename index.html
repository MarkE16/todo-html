My Todos

A small to-do app built using no frameworks/libraries (No React!). Note: Todos are saved via LocalStorage, meaning the data can be modified. This means that if it were to be modified, the program can break. Be warned. If it does happen to where messing with LocalStorage breaks, just select Clear All Todos, as this will clear LocalStorage.
